# Frontend Development Rules

## Design Skill (REQUIRED)

When creating or modifying frontend pages or components, **ALWAYS load the `frontend-design` skill first**:

```
Use the Skill tool to invoke "frontend-design"
```

This ensures:
- ✅ Distinctive, production-grade UI design
- ✅ High-quality visual polish
- ✅ Avoids generic AI aesthetics
- ✅ Consistent design patterns

## Apply Design Skill For

- Creating new pages
- Building new components
- Redesigning existing UI
- Styling or beautifying interfaces
- Adding visual features

## Tech Stack

- React 18 with JSX/TSX
- Tailwind CSS (dark theme)
- Lucide React icons
- TanStack Query for data fetching
